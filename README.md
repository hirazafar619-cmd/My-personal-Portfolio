# Portfolio Project - README.md

## **Project Summary**

This portfolio website showcases my skills as a web developer. It highlights my expertise in HTML, CSS, JavaScript, Bootstrap, and Tailwind CSS. The website includes sections such as About, Skills, Portfolio, Services, and Contact, providing a complete and visually engaging presentation of my work, capabilities, and professional identity.


## **Project Pages / Structure**

* **Type:** One-Page Website
* **Reason:** I chose a one-page structure because it allows users to scroll smoothly through all content sections in a clean and organized layout. It also creates a modern, user-friendly experience where navigation is simple and all information is accessible in one place.

## **Color Palette & Typography**

### **Color Palette:**

* **Primary Color:** `#1F2937` (Gray-800 - Sidebar)
* **Secondary Color:** `#F59E0B` (Amber/Yellow Accent)
* **Background:** `#F3F4F6` (Gray-100 – Main background)
* **Text Color:** `#1F2937` / `#4B5563` (Dark gray for paragraphs)

### **Typography / Font Scale:**

* **Font Family 1 (Headings):** Tailwind Default Sans (Inter/Roboto-like)
* **Font Family 2 (Body):** Tailwind Default Sans
* **H1 Size:** 3rem – 5rem (Responsive)
* **H2 Size:** 2rem – 3rem
* **Paragraph Size:** 1rem – 1.125rem

## **Features Implemented**

* Fully responsive layout using Tailwind CSS
* Fixed left sidebar navigation
* Hero section with background image and animated button
* About section
* Skills section with progress bars
* Portfolio grid with project images
* Services section with animated cards
* Contact form with styled inputs and button
* Footer with social icons
* AOS animation integration
